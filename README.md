
# Weather App

## Overview
A React weather application that allows users to search for weather information by city.

## Features
- Search weather by city
- Display temperature
- Display humidity
- Display wind speed
- Responsive UI
- Error handling

## Technologies Used
- React
- Vite
- JavaScript
- CSS

## Installation

npm install
npm run dev

## Known Issue

The provided API key returns an "API key is invalid" response. The application logic and API integration are implemented, and functionality can be tested using a valid API key.
